Quickstart:
  • Single Click: Collapse/Expand
  • Double Click: Zoom In/Out
  • Triple Click: Stick/Unstick Node

This visualization uses a sunburst partition layout for each individual data set provided to show the nodes in a hierarchical format. Each node can be examined individually, with detailed information appearing that contains the node's statement notation followed by its description. Checkboxes at the top left of the page determine which graphs are visible at any given time.

The nodes are color coded according to the following scheme:
  • Gold: Expanded node that contains a cross walk.
  • Orange: Collapsed node that contains a cross walk.
  • Khaki: Collapsed node that doesn't have a cross walk, but one or more of its children does.
  • Light Blue: Central node. When the graph is in an un-zoomed state, this node gives the name of the standards set.
  • Light Purple: All other expanded nodes.
  • Dark Purple: All other collapsed nodes.
The different coloring of nodes with cross walks allows the teacher to quickly identify gaps between standards, since purple nodes show that there is no alignment there. Cross walk alignments of specific nodes are shown through highlighting the aligned nodes. When a node is hovered over, it along with all nodes it shares a cross walk with are highlighted red. Additionally, a box containing information on each of its cross walks appears on the right. This allows the teacher to easily see which nodes a specific node is aligned with and view information regarding each aligned node.

Clicking a node collapses or expands it. Double clicking zooms the graph in to that node, providing a clearer view if the teacher is specifically analyzing that domain or cluster. Double clicking the center node will zoom the graph back out, following the path that it took when zooming in. Triple clicking a node will cause that node to "stick" - the node and its cross walks will remain highlighted and the cross walks information will remain visible until it is triple clicked again, increasing convenience as the mouse is free to move off of the node as the teacher examines the information and cross walks.
